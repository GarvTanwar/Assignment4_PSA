Policies

Accessibility

Powering Tomorrow should be easy to read, navigate, and understand. This page explains the accessibility approach used across the website and the improvements that should guide future updates.

Readable structure

Pages use headings, short sections, descriptive links, and consistent navigation so visitors can scan information quickly. The shared header and footer keep the main navigation in the same place across the website.

Policy pages use plain text sections instead of decorative cards so the content remains direct, readable, and easy to follow.

Keyboard navigation

The website includes a skip link for moving directly to the main content. Navigation controls are designed to be reachable by keyboard, and the mobile menu can be closed with the Escape key.

Interactive elements should keep visible focus states so keyboard users can see where they are on the page.

Images and media

Meaningful images should include alternative text that explains the purpose of the image. Decorative background visuals should be hidden from assistive technology when they do not add content.

Audio-related pages should provide a transcript or summary so visitors can access the same information without relying only on sound.

Colour, contrast, and motion

Text should have enough contrast against the background, especially in the dark footer and navigation areas. Animation should remain subtle and should not block reading or interaction.

The footer energy animation is decorative and is set behind the content. Visitors using reduced-motion settings should not be required to watch decorative motion to understand the page.

Ongoing improvements

Accessibility is an ongoing responsibility. Future updates should continue to check heading order, link text, form labels, colour contrast, mobile spacing, and keyboard behaviour before publishing changes.
